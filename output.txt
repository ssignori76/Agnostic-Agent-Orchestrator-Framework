# Test Playbook: Vulnerable 3-Layer App

## Prerequisites
- Docker and Docker Compose installed and running.

## Test Catalog

### 1. Health Check
- **ID:** TC-HEALTH
- **Pre-conditions:** All containers running.
- **Command:** `curl -f http://localhost:8080/health`
- **Expected Output:** `OK`

### 2. Item Listing
- **ID:** TC-ITEMS
- **Pre-conditions:** Backend and DB running.
- **Command:** `curl -s http://localhost:8080/api/items`
- **Expected Output:** JSON list containing 3 items.

### 3. User Registration
- **ID:** TC-REG
- **Pre-conditions:** Backend and DB running.
- **Command:** `curl -s -X POST -H "Content-Type: application/json" -d '{"username":"testuser", "password":"testpassword"}' http://localhost:8080/api/register`
- **Expected Output:** `User registered successfully`

### 4. User Login
- **ID:** TC-LOGIN
- **Pre-conditions:** User registered.
- **Command:** `curl -s -i -X POST -H "Content-Type: application/json" -d '{"username":"testuser", "password":"testpassword"}' http://localhost:8080/api/login`
- **Expected Output:** HTTP 200 and a `Set-Cookie: user_id=...` header.

### 5. Order Creation (Phase 1)
- **ID:** TC-ORDER-CREATE
- **Pre-conditions:** User logged in (cookie set).
- **Command:** `curl -s -b "user_id=1" -X POST http://localhost:8080/api/orders/create`
- **Expected Output:** JSON Order object with status `PENDING`.

### 6. Order Confirmation (Phase 2)
- **ID:** TC-ORDER-CONFIRM
- **Pre-conditions:** Order created.
- **Command:** `curl -s -b "user_id=1" -X POST http://localhost:8080/api/orders/1/confirm`
- **Expected Output:** JSON Order object with status `COMPLETED`.

### 7. Admin Access Control (Denied)
- **ID:** TC-ADMIN-DENY
- **Pre-conditions:** User logged in (not admin).
- **Command:** `curl -s -b "user_id=2" http://localhost:8080/api/admin/users`
- **Expected Output:** Error message / access denied.

### 8. Admin Access Control (Allowed)
- **ID:** TC-ADMIN-ALLOW
- **Pre-conditions:** Admin logged in (user_id=1 if admin is id 1).
- **Command:** `curl -s -b "user_id=1" http://localhost:8080/api/admin/users`
- **Expected Output:** JSON list of users.
